## Installation :

* `apt update`
* `apt install git -y`
* `git clone https://github.com/mwtutech/nexphisher.git`
* `cd nexphisher`
#### > SETUP : `bash setup`
#### > SETUP [TERMUX] : `bash tmux_setup`
#### > Run : `bash nexphisher`
